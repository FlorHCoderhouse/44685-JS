Descubriendo el DOM (Document Object Model)

Vamos a descubrir como operar sobre los elementos HTML para hacer nuestra página mucho más interesante!

Los nodos

Los elementos del DOM se organizan teniendo en cuenta las etiquetas creadas dentro de otra etiqueta (anidamiento). Esta situación determina relaciones de jerarquía entre los elementos, llamadas relaciones de padre-hijo, donde la etiqueta que contiene a otras se identifica como padre, y sus etiquetas anidadas como hijo.

Al momento de crearse el DOM se instancia un objeto, llamado nodo por cada etiqueta. Además, en la estructura existen otros nodos, cuya etiqueta no se puede identificar en el HTML: son los nodos de texto, los cuales son creados para contener las cadenas de caracteres que pueden tipearse dentro de cualquier etiqueta HTML.

Los nodos tienen diferentes tipos, entre ellos los más utilizados son los element nodes, los text nodes y los comment nodes. Podemos encontrar una lista completa en la guía de MDN.
